Experiência Profissional

Código Secreto

O projeto "Código Secreto" é uma aplicação web que permite criptografar e descriptografar texto com base em uma substituição de caracteres definida. Ele transforma letras específicas em uma sequência de caracteres correspondente e vice-versa. O objetivo principal é demonstrar um método simples de criptografia e descriptografia utilizando JavaScript, CSS e HTML.
